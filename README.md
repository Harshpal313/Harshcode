## 📌 Projects Overview

### 1. 🧮 Calculator
A simple calculator application that handles basic mathematical operations.
* **Features:** Performs addition, subtraction, multiplication, and division based on user choice.

### 2. 🔑 Password Generator
A security tool that helps users create strong, random passwords instantly.
* **Features:** Combines letters, numbers, and symbols to ensure maximum password security.

### 3. ✊✋✌️ Rock, Paper, Scissors Game
A classic text-based game where you can play against the computer.
* **Features:** Tracks scores, randomizes computer moves, and announces the winner.

## 💻 How to Run

To run these projects on your local machine, make sure you have Python installed.

1. Clone or download this repository.
2. Open your Terminal or Command Prompt in the project folder.
3. Run the specific file you want to use:
   ```bash
   python calculator.py
   ```
   *(Note: Replace `calculator.py` with the exact filename of your password generator or game file, like `password_gen.py` or `game.py`).*
